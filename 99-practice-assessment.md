## Readiness checklist

- [ ] You can describe **agent vs Copilot vs bot**.
- [ ] You can list Copilot Studio building blocks: Topics, Generative answers, Knowledge, Actions, Variables, Channels.
- [ ] You can pick architecture for: Q&A, scripted flow, action, multi-step autonomous, multi-agent.
- [ ] You can pick auth for each channel.
- [ ] You can describe Power Platform environments + DLP + managed environments.
- [ ] You can describe Solutions + PAC CLI + connection references for ALM.
- [ ] You can describe content moderation slider + RAI Impact Assessment.
- [ ] You can describe Microsoft 365 Copilot Tuning vs RAG knowledge.
- [ ] You can describe multi-agent orchestration.
- [ ] You can describe Application Insights integration + KPIs.
- [ ] You can describe CoE Starter Kit role.
- [ ] You can compute Copilot Studio capacity-unit cost.
