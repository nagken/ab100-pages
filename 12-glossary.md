# AB-100 Glossary

| Term | Meaning |
|---|---|
| **Action** | Step an agent takes - connector / Power Automate / MCP / function |
| **Adaptive card** | Rich UI element rendered in Teams / web / M365 Copilot |
| **Agent** | Autonomous AI that uses tools and takes multi-step actions |
| **ALM** | Application Lifecycle Management |
| **Application Insights** | Azure telemetry service; integrates with Copilot Studio |
| **Bot variable** | Variable persisting across conversations for one user |
| **Channel** | Where users interact with the agent (Teams, M365 Copilot, web, voice) |
| **Connection reference** | Per-environment connection abstraction |
| **Connector** | Pre-built (1500+) or custom integration with a service |
| **Content moderation slider** | Low / medium / high LLM safety setting |
| **Conversation variable** | Variable for current conversation only |
| **CoE Starter Kit** | Microsoft toolkit for Power Platform governance |
| **Copilot** | AI helper grounded on knowledge |
| **Copilot Studio** | Low-code platform to build custom agents |
| **DirectLine** | Channel API for custom apps |
| **DLP** | Data Loss Prevention; classifies connectors into Business/Non-business/Blocked |
| **Dataverse** | Power Platform's structured data store |
| **Environment** | Power Platform isolation boundary (Dev, Test, UAT, Prod) |
| **Generative answers** | LLM-grounded answers from knowledge sources (RAG) |
| **Global variable** | Variable shared across all users of an agent |
| **Knowledge source** | Where generative answers retrieve facts (SharePoint, web, OneDrive, Dataverse, custom, Graph) |
| **MCP** | Model Context Protocol - open standard for tool calls |
| **Managed environment** | Power Platform env with extra governance (solution checker, DLP, capacity) |
| **Managed solution** | Locked solution package (recommended for prod) |
| **Microsoft 365 Copilot Tuning** | Fine-tune M365 Copilot for org tone/facts |
| **Microsoft Graph connector** | Indexes 3rd-party content into M365 Search / Copilot |
| **Multi-agent orchestration** | Orchestrator agent routes to specialist agents |
| **PAC CLI** | Power Platform CLI |
| **Power Automate** | Workflow automation in Power Platform |
| **Premium connector** | Connector requiring Power Platform premium license |
| **Solution** | Package of Power Platform components to promote across envs |
| **Solution checker** | Static analysis tool in managed envs |
| **Topic** | Authored conversation flow (trigger phrases + branches) |
| **Voice channel** | Telephony / SSML-driven voice agent |
