# AB-100 Hands-On Labs

## Foundational

1. **Free Copilot Studio trial** - sign up at [copilotstudio.microsoft.com](https://copilotstudio.microsoft.com); create your first agent.
2. **PAC CLI install** - install Power Platform CLI; auth to your tenant.
3. **Power Platform admin center tour** - review environments, DLP, capacity.

## Domain 1 - Plan

4. **Create dedicated dev env** - provision a new Power Platform environment for your test agent.
5. **Set up DLP policy** - create a policy classifying connectors into Business/Non-business/Blocked.
6. **Run M365 Copilot Optimization Assessment** for your tenant; review oversharing report.

## Domain 2 - Design

7. **Build a topic** - create a "What is the return policy?" topic with branches.
8. **Add generative answers** - connect a SharePoint site as a knowledge source; ask Q&A questions.
9. **Add a connector action** - call a public REST API (e.g. weather) and surface result via adaptive card.
10. **Add a Power Automate flow** - "create a SharePoint list item" action triggered from agent.
11. **Try multi-agent orchestration** - create an orchestrator + 2 specialist agents; route by intent.
12. **Tune content moderation** - observe difference between Low / Medium / High slider settings.

## Domain 3 - Deploy

13. **Package as solution** - export your agent into a managed solution.
14. **Import to test env** - promote solution; test in test env.
15. **Connect Application Insights** - enable telemetry; query prompts + responses with KQL.
16. **Publish to Microsoft Teams** - sideload agent for your test users.
17. **Publish to M365 Copilot Chat** - register agent so it appears in M365 Copilot.
18. **Build a Power BI report** - connect to Application Insights data; visualize KPIs.
19. **Install CoE Starter Kit** - explore admin views, solution checker, capacity reports.
20. **Author an RAI Impact Assessment** - use the Microsoft template for your agent's use case.
